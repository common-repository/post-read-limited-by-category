== Installation ==

Adding Post read limited by specified category to your Wordpress site is really easy, follow these steps :

* Install and activate the this Plugin through the 'Plugins' menu in WordPress.
* After installation, click the 'PostReadLimit' item under Settings menu in the left toolbar.
* select limited category and fill in limited number, hit save button.
* Done!
